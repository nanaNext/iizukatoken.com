# 勤怠システム 事前案内（1ページ版）

## 1. 利用先
- ログイン: `https://tosouapp.com/ui/login`
- 打刻画面: `https://tosouapp.com/ui/attendance/simple`

## 2. 毎日の基本手順
1. ログインする
2. `簡易登録画面` を開く
3. 必要に応じて `勤務区分` / `勤務形態` を選ぶ
4. 出勤時に `開始打刻`
5. 退勤時に `終了打刻`
6. `開始時間 / 終了時間 / 実務時間` を確認

## 3. うまく動かない場合（1分対応）
- `開始打刻` が押せない:
  - `勤務形態` を確認
  - `再読込` を押す
  - `Ctrl + F5` を実行
  - 再ログインする
- 画面が遅い:
  - 5-10秒待つ
  - 回線（Wi-Fi/4G）を切り替える
  - 不要なタブを閉じる

## 4. 操作ルール
- 同じボタンを連打しない
- アカウントを共有しない
- 異常時はスクリーンショットを保存して管理者へ連絡

## 5. 報告時に必要な情報
- 発生時刻
- エラー画面（スクリーンショット）
- 利用端末 / ブラウザ

---
**社内向け通知文（コピー用）**  
`勤怠システムは通常稼働中です。各自のアカウントでログインし、始業時に開始打刻、終業時に終了打刻を実施してください。異常が発生した場合は、発生時刻と画面キャプチャを添えて管理者へ連絡してください。`
