# 社員向け 完全操作マニュアル（勤怠）

## 1. この資料の目的
- 本資料は、社員が日々の勤怠入力を正しく行うための手順書です。
- 対象機能:
  - ログイン
  - 簡易打刻（開始打刻 / 終了打刻）
  - 月次勤怠入力（勤務区分・現場・作業内容など）
  - エラー時の一次対応

## 2. 利用URL
- ログイン画面: `https://tosouapp.com/ui/login`
- 簡易打刻画面: `https://tosouapp.com/ui/attendance/simple`
- 月次勤怠画面: `https://tosouapp.com/ui/attendance/monthly`

## 3. 事前準備
- 推奨ブラウザ: Google Chrome（最新版）
- Edgeでログインできない/表示不具合が出る場合は、Chromeを使用してください。
- 通信環境が不安定な場合、保存や表示反映が遅れることがあります。

## 4. ログイン方法
1. ログイン画面を開く。
2. 会社から配布された `ログインID` と `パスワード` を入力する。
3. ログイン後、メニューから必要な画面を開く。

## 5. 毎日の基本運用（推奨）
1. 出勤時に `簡易登録画面` で `開始打刻` を押す。
2. 退勤時に `簡易登録画面` で `終了打刻` を押す。
3. 当日または月内に `月次勤怠画面` で内容を確認・補足入力する。
4. 必要に応じて `現場`・`作業内容`・`理由` などを入力し、保存する。

## 6. 簡易登録画面の使い方
- `開始打刻`:
  - 出勤時に1回だけ押してください。
  - `開始打刻済み` の表示後は、同画面で再打刻できません。
- `終了打刻`:
  - 退勤時に押してください。
- 注意:
  - 同じボタンを連打しないでください。
  - 時刻修正が必要な場合は、月次勤怠画面で修正してください。

## 7. 月次勤怠画面の使い方
### 7.1 入力できる主な項目
- 勤務区分（例: 出勤、休日出勤、半休、休日、有給休暇）
- 現場
- 作業内容
- 開始時間 / 終了時間
- 休憩時間
- 理由（必要時）

### 7.2 保存の基本
- 行を変更すると自動保存される場合があります。
- 画面上部/下部の保存ボタンでも保存できます。
- 保存後に画面を再読込し、内容が反映されていることを確認してください。

### 7.3 重要ルール
- `出勤` や `休日出勤` を選択した場合、必要に応じて時刻も確認してください。
- 打刻漏れがある場合でも、月次入力内容は保存できます。
- ただし「実打刻」と「月次入力」は別データのため、運用ルールに従って補正してください。

## 8. 作業報告画面の見方（社員向け理解）
- 状態表示は、打刻有無と入力内容で変わります。
- 主な状態の意味:
  - `勤務中`: 当日、開始のみで終了未打刻
  - `退勤漏れ`: 過去日で終了未打刻
  - `退勤漏れ（入力済み）`: 終了未打刻だが月次入力は済み
  - `未提出`: 打刻済みだが報告内容不足
  - `提出済`: 必要入力が揃っている
  - `月次入力済み（打刻なし）`: 打刻なしだが月次入力済み

## 9. スマートフォン利用について
- 社員画面はスマートフォンでも利用可能です。
- ただし、管理画面はスマートフォン最適化が未対応のため、管理操作はPC利用を推奨します。

## 10. よくあるトラブルと対処
### 10.1 保存したのに戻る/変わらない
- `Ctrl + F5` で強制再読込
- 再ログイン
- Chromeで再確認

### 10.2 打刻ボタンが押せない
- すでに `開始打刻済み` の可能性があります。
- 同日の再修正は月次勤怠画面で行ってください。

### 10.3 画面が遅い/反映が遅い
- 数秒待機してから再確認
- 通信環境（Wi-Fi/4G）を切り替え
- 不要なタブを閉じる

## 11. 禁止事項
- アカウントの共有
- 他人のID/パスワード利用
- エラー時に連打で回避しようとする操作

## 12. 問い合わせ時に必ず送る情報
- 発生日時
- 対象画面URL
- スクリーンショット
- 利用端末（PC/スマホ）とブラウザ（Chrome/Edge等）
- 実施した操作手順（何を押したか）
