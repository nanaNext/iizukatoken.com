# マネージャー向け 運用操作マニュアル（実務版）

## 1. 本書の目的
- 本書は、マネージャーが日常運用で必要な操作を迷わず実施できるようにする手順書です。
- 対象:
  - ログイン
  - 社員作成（社員管理）
  - 勤怠確認（月次集計・作業報告）
  - 打刻漏れ対応
  - よくあるエラー時の一次対応

## 2. 利用前の重要注意
- ログインURLは社員と同じです（共通ログイン画面）。
- ただし、マネージャー権限で操作するには、必ずマネージャー用アカウントでログインしてください。
- 管理画面はスマートフォン最適化が未対応のため、PC利用を推奨します。
- Edgeで不安定な場合は、Google Chromeを利用してください。

## 3. ログイン手順（マネージャー）
1. ログイン画面を開く: `https://tosouapp.com/ui/login`
2. マネージャー用 `ユーザーID / パスワード` を入力する。
3. ログイン後、上部メニューから `社員管理`・`勤怠管理` を開く。

## 4. 社員作成手順（社員管理）
### 4.1 新規作成
1. 上部メニュー `社員管理` を開く。
2. `新規`（または社員追加ボタン）を押す。
3. 必須情報を入力する:
   - 氏名
   - 生年月日
   - メールアドレス（本人の実メール）
   - 部署、入社日など必要項目
4. `保存` / `更新` を押す。
5. 一覧に戻り、対象社員が追加されたことを確認する。

### 4.2 ログイン情報の運用ルール
- ログインID: 本人メールアドレス
- 初期パスワード: 生年月日（`YYYYMMDD`）
  - 例: 1998年4月5日 -> `19980405`
- 初回ログイン後、本人へパスワード変更を案内してください。

### 4.3 社員情報の修正
1. `社員管理` 一覧で対象社員の `編集` を押す。
2. 必要項目を修正する。
3. `更新` を押す。
4. 再表示して修正内容が反映されたことを確認する。

## 5. 個人書類画像（社員管理）
1. `社員管理` -> 対象社員 `編集` を開く。
2. `個人書類画像（アップロード）` でファイルを選択する。
3. `更新`（またはアップロード）を実行する。
4. `保存済み写真` に表示されることを確認する。

注意:
- この画像は「顔アイコン用」ではなく、社員の書類画像管理用です。
- 画像名やURLで古いデータが表示される場合は再アップロードしてください。

## 6. 月次集計の確認・入力（管理者入力）
1. 上部メニュー `勤怠管理` -> `月次集計` を開く。
2. `社員` と `対象年月` を選ぶ。
3. `読込` を押して対象データを表示する。
4. 必要に応じて以下を確認・入力:
   - 月次サマリ（管理者入力）
   - シフト割当（管理者入力）
   - 契約先一覧/契約内容/業務内容（管理者入力）
5. `保存` を押す。
6. `読込完了` 表示、または再読込で内容保持を確認する。

## 7. 作業報告の確認（ステータスの意味）
- `勤務中`: 当日、開始打刻のみ
- `退勤漏れ`: 過去日で終了打刻なし
- `退勤漏れ（入力済み）`: 終了打刻なしだが月次入力あり
- `月次入力済み（打刻なし）`: 打刻なしだが月次入力あり
- `未提出`: 打刻済みだが報告内容不足
- `提出済`: 必要入力が揃っている

運用ポイント:
- 退勤漏れ表示の社員には、月次入力だけでなく打刻ルールを再案内してください。
- 実打刻と月次入力は別管理のため、両方確認することが重要です。

## 8. 1日の運用フロー（推奨）
### 始業前
- マネージャーアカウントでログイン確認
- 必要なら当日の対象社員を確認

### 日中
- 作業報告で異常ステータス（退勤漏れ、未提出）を確認
- 社員へ入力・修正依頼

### 終業後
- 当日分の未提出/退勤漏れを一覧確認
- 翌日フォロー対象を整理

## 9. よくあるトラブルと対応
### 9.1 保存したのに戻る
- `Ctrl + F5` で強制再読込
- 再ログイン
- Chromeで再確認

### 9.2 Forbidden 表示
- 権限対象外社員を開いていないか確認
- いったん社員選択を変更して再読込
- 継続時はシステム担当へ連絡

### 9.3 画像が表示されない
- 一度再アップロード
- 画像サイズ/形式を変更して再試行
- ブラウザキャッシュをクリア

## 10. 問い合わせ時に必要な情報
- 発生日時
- 対象社員番号
- 対象画面（社員管理/月次集計/作業報告など）
- エラーメッセージ
- スクリーンショット
- 利用ブラウザ（Chrome/Edge）と端末（PC/スマホ）

## 11. 社員への連絡テンプレート（短文）
`本日の勤怠入力をご確認ください。開始・終了打刻、月次入力（勤務区分/現場/作業内容）に不足がある場合は本日中に修正をお願いします。`
